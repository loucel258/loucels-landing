Loucels Widget Test Page

If everything is wired correctly, you should see the chat launcher in the bottom-right corner.

What to check

  • Launcher button appears bottom-right with your brand color.
  • Click it — chat panel opens with the greeting message.
  • Send a message — agent responds.
  • Open DevTools → Network — confirm CORS headers on /api/agent/loucels-landing/chat.
Dev only — never ship this to public/. Serve it with cd scripts/dev && python3 -m http.server 8080 and open http://localhost:8080/test-widget.html. The dev agent slug (loucels-landing-dev) allows localhost origins; the production slug does not.
